# remote-tara

太良リモートワークセンター（仮称）の紹介ページです。

## 公開ファイル

- `index.html`: GitHub Pages で公開するトップページ
- `.nojekyll`: GitHub Pages を Jekyll 変換なしで配信するための設定ファイル

## GitHub Pages で公開する手順

1. GitHub で `remote-tara` という新しいリポジトリを作成します。
2. このフォルダ内の `index.html` と `.nojekyll` をリポジトリ直下に置きます。
3. GitHub の `Settings` > `Pages` を開きます。
4. `Build and deployment` で `Deploy from a branch` を選びます。
5. ブランチを `main`、フォルダを `/ (root)` に設定して保存します。

数分後に GitHub Pages の公開URLでページが表示されます。
